Este programa resuelve el problema de la Torre de Hanoi utilizando una estructura orientada a objetos y ejecutando los movimientos en paralelo con hilos. Define dos clases: Disco, que representa cada disco con un tamaño, y Torre, que maneja una pila de discos y permite agregar o quitar uno respetando las reglas del juego. La función mover_disco realiza el movimiento entre torres validando que sea legal. La función resolver_hanoi implementa la solución recursiva clásica, pero en lugar de ejecutar los movimientos directamente, envía cada uno como una tarea a un ThreadPoolExecutor, lo que permite paralelizar la ejecución. Cada movimiento, al ejecutarse, imprime el cambio realizado. Además, se usa una función simular_carga para introducir un retardo aleatorio y simular carga de procesamiento. El programa solicita al usuario cuántos discos usar, inicializa las tres torres colocando todos los discos en la primera y muestra el estado inicial. Luego crea un grupo de tres hilos para procesar los movimientos y espera a que todas las tareas terminen. Al final, muestra cómo quedaron las torres y el tiempo total de ejecución.
